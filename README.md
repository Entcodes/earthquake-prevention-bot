# 防震问答机器人 | Earthquake Prevention Q&A Bot

一个功能完整的中英双语地震防灾知识问答机器人，提供地震前兆识别、历史案例查询、应急指南和智能问答功能。

## 🌍 项目特点

- **暗色主题设计**：专业美观的深色UI界面
- **中英双语支持**：一键切换中文/英文界面
- **四大核心模块**：
  - 🔍 地震前兆识别（动物/天空/建筑物 + 三级警告）
  - 📋 地震以往案例（中国10大案例 + 国际对比）
  - 📖 地震须知（黄金法则、应急包清单、自救互救）
  - 🤖 问答机器人（关键词触发 + 大学生贡献建议）
- **图像识别功能**：模拟AI分析上传的图片，检测地震前兆特征
- **纯前端实现**：单HTML文件，无需服务器，可直接浏览器打开

## 🚀 快速开始

### 方法1：直接打开（推荐）
1. 下载 `index.html` 文件
2. 用浏览器（Chrome/Firefox/Edge）直接打开即可使用

### 方法2：本地服务器
```bash
# 如果有Python
python -m http.server 8080

# 或者用Node.js
npx serve .
```
然后访问 `http://localhost:8080`

## ✨ 功能详情

### 1. 地震前兆识别
- **动物异常**：58+种动物可感知地震前兆（狗、家禽、牲畜、鼠类、蛇类等）
- **天空异象**：地震光、异常云层、地气地雾等
- **建筑物反应**：墙体裂缝、门窗变形、地下水异常、地声等
- **三级警告系统**：低风险（绿色）、中风险（黄色）、高风险（红色）
- **免责声明**：明确说明信息仅供参考，不构成地震预测

### 2. 地震以往案例
- **中国10大地震**：汶川8.0、玉树7.1、芦山7.0、泸定6.8、积石山6.2等
- **国际对比**：日本（311防灾体系）、新西兰（隔震技术）、美国（加州抗震规范）
- 每个案例包含：时间、震级、伤亡情况、应急处置、重建措施、经验教训

### 3. 地震须知
- **关键数据**：黄金避险时间10-20秒、黄金救援72小时、存活率>90%
- **黄金法则**：伏地、遮挡、手抓牢（Drop, Cover, Hold On）
- **震前准备**：家庭应急预案、逃生路线、应急演练
- **应急包清单**：水、食物、医疗用品、通讯照明、工具文件
- **自救互救**：被困自救、五先五后原则、骨折急救

### 4. 问答机器人
- **12个知识库主题**：地震避险、大学生贡献、应急包、前兆识别、自救、避难、黄金时间、治理建议等
- **智能关键词匹配**：自动识别用户问题并返回相关答案
- **大学生贡献专题**：三阶段贡献建议（防灾前置、救灾支援、灾后重建）
- **快捷按钮**：点击即可快速提问

### 5. 图像识别（新增）
- **拖放上传**：支持JPG/PNG/WEBP格式，最大10MB
- **模拟AI分析**：2.5秒加载动画，三阶段状态提示
- **检测结果**：随机展示2-4个前兆特征，含置信度进度条
- **风险评估**：综合风险等级（低/中/高）+ 行动建议
- **教育性质**：明确标注为模拟演示，非真实预测

##  技术栈

##  自定义修改

### 修改颜色主题
编辑 `index.html` 中的 CSS 变量（第9行）：
```css
:root {
  --bg0: #08080e;      /* 最深背景 */
  --acc: #4a9eff;      /* 主色调 */
  --red: #ff4a6a;      /* 高风险色 */
  --org: #ffb84a;      /* 中风险色 */
  --grn: #4aff8a;      /* 低风险色 */
  /* ... 其他变量 */
}
```

### 添加新的问答内容
在 JavaScript 的 `kb` 数组中添加新条目（约第450行）：
```javascript
{
  keys: ['关键词1', '关键词2'],
  cn: '中文回答内容',
  en: 'English answer content'
}
```

### 修改前兆数据库
编辑 `precursorDB` 数组（约第440行），添加或修改前兆项。
