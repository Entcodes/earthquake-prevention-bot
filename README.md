# 防震问答机器人技能 | Earthquake Q&A Bot Skill

一个专业的地震防灾知识库技能，为 Qoder AI 提供地震前兆识别、避险自救、应急准备、案例分析等全面的防灾指导能力。

##  功能特性

- 🔍 **地震前兆识别** - 动物异常、天空异象、建筑物反应 + 三级风险评估
- 🛡️ **避险黄金法则** - Drop, Cover, Hold On + 各场景应对指南
- 🎒 **应急包准备** - 完整物品清单和储备建议
- 🆘 **自救互救** - 被困六步法、互救五先五后、常见伤害急救
- 📋 **历史案例** - 中国10大地震 + 日/新/美国际对比
- 🎓 **大学生贡献** - 三阶段贡献建议 + 学校/政府政策改进
- 🏛️ **治理建议** - 学校/政府政策改进方案
- 🌐 **中英双语** - 支持中英文关键词触发

##  使用方法

### 在 Qoder 中使用

此技能会自动在以下场景触发：
- 用户询问地震相关问题（地震、前兆、避险等）
- 用户询问灾害预防和应急准备
- 用户询问自救互救方法
- 用户询问地震历史案例

### 关键词触发

**中文关键词**：地震、前兆、动物、异常、自救、避难、应急包、黄金时间、大学生、贡献、治理、案例

**English Keywords**: earthquake, shake, precursor, rescue, shelter, kit, student, governance, emergency, disaster prevention

##  知识库内容

### 1. 地震前兆识别
- 动物异常（58+种动物）：家畜家禽、野生动物
- 天空异象：地震光、异常云层、震前闷热、地气地雾
- 建筑物反应：墙体裂缝、门窗变形、地下水异常、地声、悬挂物摆动
- 三级警告系统：低风险（绿）、中风险（黄）、高风险（红）

### 2. 地震避险黄金法则
- **伏地、遮挡、手抓牢** (Drop, Cover, Hold On)
- 8种场景应对：室内/户外/开车/商场/电梯/床上/山区/海岸
- 关键时间数据：黄金避险10-20秒、黄金救援72小时

### 3. 应急包清单（至少3天量）
- 生存保障：水、食物
- 医疗防护：急救箱、药品
- 通讯照明：手摇收音机、手电筒
- 工具文件：多功能刀、绳索

### 4. 震后自救互救
- 被困自救六步法
- 互救"五先五后"原则
- 常见伤害急救要点

### 5. 地震案例
- **中国10大地震**：汶川8.0、玉树7.1、芦山7.0等
- **国际经验**：日本（防灾公园）、新西兰（隔震技术）、美国（抗震规范）

### 6. 大学生贡献
- 防灾前置准备
- 救灾应急支援
- 灾后长期重建调研

### 7. 治理建议
- 学校政策改进
- 政府政策改进

### 8. 灾后恢复
- 安全隐患排查
- 心理援助
- 重建政策与资金

## ⚠️ 免责声明

本技能提供的地震信息仅供参考，**不构成任何地震预测或预警**。地震预测目前仍属于世界性科学难题，任何单一前兆现象都不能作为确定地震即将发生的依据。请以中国地震局及各地地震监测部门发布的官方信息为准。遇到紧急情况，请拨打当地应急电话（中国：12322防震减灾服务热线 / 119消防 / 120急救）。
